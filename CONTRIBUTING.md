# Contributing Guide

Thank you for considering contributing to the Entra ID Assessment project!

## How to Contribute
1. Fork the repository on GitHub.
2. Create a feature branch (e.g., `feature/add-template`).
3. Make your changes (follow the structure in `README.md`).
4. Test your scripts or templates before submitting a pull request.
5. Submit a pull request with a clear description of your changes.

## Code Style & Structure
- Follow PowerShell best practices (lint with `PSScriptAnalyzer` where possible).
- Keep documentation in Markdown.
- Use Microsoft Graph APIs for read-only data collection.
- Respect customer privacy and GDPR policies when handling assessment data.

## Attribution
Wherever you adapt or quote Microsoft’s content, include an attribution line referencing Microsoft’s original Azure AD Assessment project.
